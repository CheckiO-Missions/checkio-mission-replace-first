In a given array first element should become the last one. Empty array or array with only one element should stay the same

Input: {% if interpreter.slug == "js-node" %}Array{% else %}List{% endif %}.

Output: {% if interpreter.slug == "js-node" %}Array{% else %}List{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
replaceFirst([1, 2, 3, 4]) == [2, 3, 4, 1]
replaceFirst([1]) == [1]
{% else %} 
replace_first([1, 2, 3, 4]) == [2, 3, 4, 1]
replace_first([1]) == [1]
{% endif %}
