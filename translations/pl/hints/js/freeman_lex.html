chodzi o to, aby usunąć pierwszy element z tablicy, zachować go i dodać na koniec tablicy.

Ok, od czego powinienem zacząć?

Przede wszystkim proszę zrozumieć, że jakiekolwiek operacje należy wykonywać tylko wtedy, gdy tablica ma co najmniej dwa elementy. W przeciwnym razie można natychmiast zwrócić tablicę.

Aby znaleźć długość tablicy, proszę użyć właściwości Array.length. 

if (values.length> 1)
    ...;

return values;

Ponieważ wewnątrz instrukcji if znajduje się tylko jeden wiersz kodu, nie ma potrzeby używania nawiasów klamrowych.

Ok, co robić dalej?

Proszę usunąć i zapisać pierwszy element tablicy. Proszę użyć metody Array.shift(). Usuwa ona i zwraca (zapisuje) pierwszy element. 

values.shift()

Jak umieścić ten element na końcu tablicy?

Do tego celu przyda się inna metoda - Array.push(). Ponieważ metoda Array.shift() zwraca wartość, mogą Państwo użyć jej jako argumentu dla metody Array.push(). 

values.push(values.shift());

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę po prostu opakować swój kod w wyrażenie funkcyjne.

Oto całe rozwiązanie:

function replaceFirst(values: number[]): number[] {

    if (values.length> 1)
        values.push(values.shift());
    
    return values;
}

Jasne, proszę pobrać!

W tym rozwiązaniu używane są inne metody: Array.slice(index) - aby uzyskać nową tablicę, zaczynając od potrzebnego elementu, Array.concat() - aby połączyć tablice. Użyto tu również operatora trójargumentowego, który jest inną formą zapisu warunku. 

function replaceFirst(values: number[]): number[] {

    return values.length> 1? values.slice(1).concat(values[0]): values;
    // or
    return values.length> 1? [...values.splice(1), values[0]]: values; 
}

To rozwiązanie jest skomplikowane, ale oznacza to samo: jeśli v.length - proszę wykonać operacje, po zwróceniu v, jeśli nie - proszę po prostu zwrócić v. 

function replaceFirst(v: number[]): number[] {
    
    return v.length && v.push(v.shift()), v;
}
