我们的想法是从数组中删除第一个元素，保留它并添加到数组的末尾。

好吧，我应该从什么开始呢？

首先，你需要明白，只有当数组至少有两个元素时，你才需要执行任何操作。否则，你可以立即返回数组。

要查找数组的长度，请使用Array.length 属性。 

if (values.length> 1)
    ...;

return values;

因为在if 语句中只有一行代码，所以不需要使用大括号。

好了，下一步做什么？

你需要删除并存储数组的第一个元素。使用Array.shift() 方法。它删除并返回（存储）第一个元素。 

values.shift()

如何将该元素放在数组的末尾？

为了这个目的，另一个方法将是有用的 -Array.push() 。因为Array.shift() 返回值，你可以用它作为Array.push() 方法的参数。 

values.push(values.shift());

那么，我的最终解决方案应该是怎样的呢？

您只需要将您的代码封装成函数表达式。

下面是整个解决方案：

function replaceFirst(values: number[]): number[] {

    if (values.length> 1)
        values.push(values.shift());
    
    return values;
}

在这个解决方案中使用了其他方法： Array.slice(index) - 得到一个新的数组，从需要的元素开始，Array.concat() - 连接数组。这里还使用了三元运算符，这是写条件的另一种形式。 

function replaceFirst(values: number[]): number[] {

    return values.length> 1? values.slice(1).concat(values[0]): values;
    // or
    return values.length> 1? [...values.splice(1), values[0]]: values; 
}

这个解决方法比较麻烦，但意思是一样的：如果v.length - 进行操作，然后返回v ，如果不是 - 只返回v 。 

function replaceFirst(v: number[]): number[] {
    
    return v.length && v.push(v.shift()), v;
}
