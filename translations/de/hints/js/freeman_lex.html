💡 Die Idee ist, das erste Element aus dem Array zu entfernen, es zu behalten und am Ende des Arrays hinzuzufügen.

Ok, womit soll ich beginnen?

Zunächst einmal müssen Sie verstehen, dass Sie nur dann eine Operation durchführen müssen, wenn das Array mindestens zwei Elemente enthält. Andernfalls können Sie das Array sofort zurückgeben.

Um die Länge des Arrays zu ermitteln, verwenden Sie die Eigenschaft Array.length. 

if (values.length> 1)
    ...;

return values;

Da Sie nur eine Codezeile innerhalb der Anweisung if haben, müssen Sie keine geschweiften Klammern verwenden.

Ok, was machen wir als nächstes?

Sie müssen das erste Element des Arrays entfernen und speichern. Verwenden Sie die Methode Array.shift(). Sie entfernt das erste Element und gibt es zurück (speichert es). 

values.shift()

Wie können Sie dieses Element am Ende des Arrays ablegen?

Für diesen Zweck ist eine andere Methode nützlich - Array.push(). Da Array.shift() einen Wert zurückgibt, können Sie diesen als Argument für die Methode Array.push() verwenden. 

values.push(values.shift());

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen Ihren Code nur in einen Funktionsausdruck verpacken.

Hier ist die gesamte Lösung:

function replaceFirst(values: number[]): number[] {

    if (values.length> 1)
        values.push(values.shift());
    
    return values;
}

In dieser Lösung werden andere Methoden verwendet: Array.slice(index) - um ein neues Array zu erhalten, beginnend mit dem benötigten Element, Array.concat() - um Arrays zu verbinden. Außerdem wird hier der ternäre Operator verwendet, eine andere Form, um Bedingungen zu schreiben. 

function replaceFirst(values: number[]): number[] {

    return values.length> 1? values.slice(1).concat(values[0]): values;
    // or
    return values.length> 1? [...values.splice(1), values[0]]: values; 
}

Diese Lösung ist kompliziert, bedeutet aber dasselbe: wenn v.length - führen Sie die Operationen aus, danach geben Sie v zurück, wenn nicht - geben Sie einfach v zurück. 

function replaceFirst(v: number[]): number[] {
    
    return v.length && v.push(v.shift()), v;
}
