Ok, what should I start from?

First of all, you need to understand, that you only need to perform any operations, if the Array has at least two elements. Otherwise, you may return the Array immediately.

To find the length of the Array, use Array.length property. 

if (values.length > 1)
    ...;

return values;

Since you will have only one line of code inside if statement, no need to use curly brackets.

Ok, what to do next?

You need to remove and store the first element of the Array. Use Array.shift() method. It removes and returns (store) the first element. 

values.shift()
How to put this element at the end of the Array?

For this purpose another method will be useful - Array.push(). Since, Array.shift() returns value, you may use it as the argument for Array.push() method. 

values.push(values.shift());
So, how my final solution should look like?

You just need to wrap your code into function expression.

Here is the whole solution:

function replaceFirst(values: number[]): number[] {

    if (values.length > 1)
        values.push(values.shift());
    
    return values;
}

function replaceFirst(values: number[]): number[] {

    return values.length > 1 ? values.slice(1).concat(values[0]) : values;
    // or
    return values.length > 1 ? [...values.splice(1), values[0]] : values;
